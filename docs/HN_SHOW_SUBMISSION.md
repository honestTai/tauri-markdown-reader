## First comment after submission

I built Markdown Reader, a local-first desktop app for reading project Markdown folders and turning PDF/DOCX files into editable Markdown drafts.

It is not another writing-first Markdown editor. The goal is to open an existing project folder and quickly read/search PRDs, README files, technical plans, meeting notes, and troubleshooting logs without opening an IDE.

Features:
- Open a folder or single Markdown file
- Full-text search across filenames, titles, frontmatter, and body text
- Outline navigation and scroll-position restore
- Favorites, pinned docs, and quick open
- Light editing, local image insertion, dropped images, and pasted screenshots
- PDF to Markdown draft import
- DOCX to Markdown draft import with images saved into `.assets`
- Export to Word, PDF, Markdown, plain text, and reading HTML
- Bundled `md-reader` CLI for local agent workflows
- Tauri + React + TypeScript
- Windows and macOS Apple Silicon / M-series builds
